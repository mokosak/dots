# dots

small suckless setup.

![screenshot](screenshot.png)

theme is ayu mirage with a green accent.

## keybinds (mod = super)

| key | does |
|-----|------|
| `mod+enter` | terminal |
| `mod+d` | launcher |
| `mod+b` | browser |
| `mod+m` | music (ncmpcpp) |
| `mod+y` | youtube search (ytfzf) |
| `mod+w` | wallpaper picker |
| `mod+x` | lock |
| `mod+space` | toggle layout |
| `mod+j`/`k` | focus down/up |
| `mod+h`/`l` | shrink/grow master |
| `mod+z` | zoom to master |
| `mod+shift+f` | fullscreen |
| `mod+q` / `mod+shift+q` | kill window / quit dwm |
| `mod+shift+e` | exit menu |
| `mod+[1-9]` | tags |
| `mod+shift+s` | area screenshot |
| `print` / `shift+print` | full screenshot / screenshot menu |
| media keys | volume · brightness · playback |

## status bar

made using dwmblocks - its clickable.

## install

```sh
git clone https://github.com/mokosak/dots
cd dots
./install.sh
```
install script also backs up previous conf files.
